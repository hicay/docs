Congratulations, you’re on your way to using TranscribeTube API to build something awesome! We offer a range of APIs to suit your needs.
Whether you’re looking for a simple API to get transcription from youtube videos or want to transcribe your video and audio files in any language, we’ve got you covered.

## Prerequisite

Our API is available to Personal Plan, Basic Plan users and Pro Plan users. Once you subscribe one of these plans, you can follow [Get your API key](/api-documentation/authentication) to start testing right away.

## Getting Help

If you have any questions, please email us at hello@transcribetube.com or get help in chat in [App](https://app.transcribetube.com/dashboard).
